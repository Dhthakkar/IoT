Street Road Reviewing System

Project Objective :


This project is based on IOT. The Road Reviewing System will be used to measure Smoothnesss/Rughness of road , pollution , light intensity and trafic of road.


Features :

        

  • To measure Smoothness/Roughness of Road : Sensor used is Gyroscope

    •     

  • To measure Pollution of Road : Sensor used is MQ135 Gas Sensor

    •     

  • To measure Trafic on Road : Sensor used is Ultrasonic Sensor

    •     

  • To measure light intensity on Road : Sensor used is LDR Sensor


Details :

    

Various sensors will provide various readings at particular time span which will be sent to android application named 'Blue Term' by arduino . These readings will be saved in log file generated by blue term and then this log file will be sent to web server where these readings will be parsed and according to that graph will be generated and various graphical meter will display readings of sensors over time.
